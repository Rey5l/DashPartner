# Система модерации ресурсов

## Описание

Реализована система модерации для добавления ресурсов (каналов) пользователями. Теперь при добавлении нового ресурса он отправляется на модерацию администраторам.

## Что было реализовано

### 1. База данных

Обновлена таблица `user_resources`:
- `owner_username` - username пользователя
- `category` - категория ресурса
- `status` - статус модерации (pending/approved/declined)
- `moderated_at` - дата модерации
- `moderated_by` - ID администратора, который модерировал

### 2. Процесс добавления ресурса

1. Пользователь добавляет ресурс через "Мои ресурсы" → "Добавить ресурс"
2. Ресурс сохраняется со статусом `pending`
3. Пользователь получает уведомление, что ресурс отправлен на модерацию
4. Ресурс появляется в админ панели в разделе "Модерация"

### 3. Админ панель - Модерация

Новая вкладка "✅ Модерация" в админ панели:

#### Список ресурсов на модерации
- Показывает все ресурсы со статусом `pending`
- Отображает: ID ресурса, username пользователя, название ресурса
- Счетчик количества ресурсов на модерации

#### Просмотр ресурса
При выборе ресурса показывается:
- ID и username пользователя
- Название ресурса
- Ссылка на ресурс
- Дата создания
- Кнопки: "✅ Принять" и "❌ Отклонить"

#### Принятие ресурса
1. Админ нажимает "✅ Принять"
2. Появляется клавиатура с выбором категории (34 категории)
3. После выбора категории:
   - Ресурс получает статус `approved`
   - Устанавливается выбранная категория
   - Пользователь получает уведомление об одобрении
   - Ресурс становится доступен для использования в конкурсах

#### Отклонение ресурса
1. Админ нажимает "❌ Отклонить"
2. Ресурс получает статус `declined`
3. Пользователь получает уведомление об отклонении
4. Ресурс не появляется в списке доступных ресурсов

## Использование

### Для пользователей
1. Перейти в "Мои ресурсы"
2. Нажать "Добавить ресурс"
3. Отправить ссылку на канал
4. Дождаться модерации

### Для администраторов
1. Открыть админ панель (`/admin` или кнопка "🔐 Админ панель")
2. Выбрать "✅ Модерация"
3. Выбрать ресурс из списка
4. Нажать "✅ Принять" и выбрать категорию, или "❌ Отклонить"

## Примечания

- Пользователи видят только одобренные ресурсы (`status = 'approved'`)
- Отклоненные ресурсы не удаляются из базы, но не отображаются пользователю
- При одобрении/отклонении пользователь получает уведомление
- Категория устанавливается только при одобрении ресурса
