# Документация: Графики статистики

## Описание
Добавлена функциональность генерации графиков статистики за выбранный период (10, 20 или 30 дней) для различных разделов бота.

## Добавленные разделы с графиками

### 1. Общая статистика системы (Админ панель)
- **Путь**: Админ панель → Статистика
- **Callback**: `admin_statistics`
- **Кнопки**: 
  - 📊 10 дней (`admin_stats_graph_10`)
  - 📊 20 дней (`admin_stats_graph_20`)
  - 📊 30 дней (`admin_stats_graph_30`)
- **График**: Показывает количество новых пользователей за период

### 2. Статистика ресурсов (Админ панель)
- **Путь**: Админ панель → Ресурсы → Статистика
- **Callback**: `admin_resource_stats`
- **Кнопки**:
  - 📊 10 дней (`resource_stats_graph_10`)
  - 📊 20 дней (`resource_stats_graph_20`)
  - 📊 30 дней (`resource_stats_graph_30`)
- **График**: Показывает количество новых ресурсов за период

### 3. Статистика конкурсов
- **Путь**: Продать трафик → Статистика
- **Callback**: `contest_stats`
- **Кнопки**:
  - 📊 10 дней (`contest_stats_graph_10`)
  - 📊 20 дней (`contest_stats_graph_20`)
  - 📊 30 дней (`contest_stats_graph_30`)
- **График**: Показывает количество участников конкурсов за период

## Технические детали

### Новые файлы

1. **services/chart_generator.py**
   - Функция `generate_statistics_chart()` - генерирует график
   - Функция `cleanup_old_charts()` - удаляет старые графики (старше 24 часов)
   - Использует matplotlib для создания графиков
   - Сохраняет графики в `data/charts/`

2. **services/database_service_stats.py**
   - Класс `DatabaseStatsService` с методами для получения статистики за период:
     - `get_users_stats_by_period()` - новые пользователи
     - `get_earnings_stats_by_period()` - заработок
     - `get_resources_stats_by_period()` - новые ресурсы
     - `get_contests_stats_by_period()` - участники конкурсов
     - `get_chat_access_stats_by_period()` - одобренные пользователи
     - `get_withdrawals_stats_by_period()` - выводы средств

     - `resource_stats_graph_callback()` - генерация графика статистики ресурсов

## Как это работает

1. Пользователь открывает раздел статистики
2. Видит текстовую статистику и кнопки выбора периода (10, 20, 30 дней)
3. Нажимает на кнопку с нужным периодом
4. Бот:
   - Получает данные из базы данных за выбранный период
   - Генерирует график с помощью matplotlib
   - Сохраняет график в `data/charts/`
   - Отправляет фото с графиком пользователю
   - Удаляет старые графики (старше 24 часов)

## Формат графиков

- **Размер**: 12x6 дюймов (1800x900 пикселей при 150 DPI)
- **Формат**: PNG
- **Стиль**: Линейный график с заливкой, точками и подписями значений
- **Оси**: 
  - X - даты в формате DD.MM
  - Y - количество (автоматическое масштабирование)
- **Цвет**: Синий (#2196F3)

## Автоматическая очистка

Старые графики (старше 24 часов) автоматически удаляются при генерации нового графика для экономии места на диске.

## Примечания

- Графики генерируются в реальном времени при каждом запросе
- Если за период нет данных, отображается пустой график с нулевыми значениями
- Поддержка кириллицы в подписях графиков
- Графики сохраняются с уникальными именами (включают тип, период и timestamp)

## 4. Статистика конкретного чата

### Описание
Добавлена возможность просмотра статистики для каждого конкретного чата с генерацией графиков.

### Как использовать
1. Перейдите в раздел **Продать трафик → Чаты**
2. Выберите нужный чат из списка
3. Нажмите кнопку **📊 Статистика**
4. Выберите период (10, 20 или 30 дней)
5. Бот сгенерирует и отправит график

### Что показывает график
- Количество одобренных пользователей в чате по дням
- Динамика роста аудитории за выбранный период

### Технические детали

**Изменённые файлы:**

1. **keyboards/main_keyboard.py**
   - Добавлена кнопка "📊 Статистика" в `get_chat_settings_keyboard()`

2. **routes/main_routes.py**
   - Добавлен обработчик `chat_stats_callback()` - показывает статистику чата с кнопками выбора периода
   - Добавлен обработчик `chat_stats_graph_callback()` - генерирует график для конкретного чата

3. **services/database_service.py**
   - Добавлен метод `count_approved_members()` - подсчитывает количество одобренных пользователей в чате

4. **services/database_service_stats.py**
   - Добавлен метод `get_chat_stats_by_period()` - получает статистику одобренных пользователей для конкретного чата за период

### Callback данные
- `chat_stats:{chat_id}` - открыть статистику чата
- `chat_stats_graph:{chat_id}:{period}` - сгенерировать график для чата

### Пример использования
```
Пользователь → Продать трафик → Чаты → [Выбрать чат] → 📊 Статистика → 📊 20 дней
```

Бот отправит график с подписью:
```
📊 Статистика одобренных пользователей
💬 Чат: Название чата
📅 Период: 20 дней
```
